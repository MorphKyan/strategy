## 0. 课题定位与研究依据

### 0.1 定位：核心-卫星结构里的卫星仓

用户现有实盘是 R038 固定权重 + 5/25 阈值带的永久组合（低波纪律型，组合 `live_r8_permanent_100k`）。本课题为**独立的高波卫星仓**新开一条策略线：行业 ETF 横截面动量轮动。

三条纪律先立住：

1. **世界观切割**：轮动本质是横截面择时，与 R038 的"择时不关键"哲学互斥。两者在**组合层**完全隔离——独立配置、独立组合 ID（`sim_r9_*` / `live_r9_*`）、独立归因，卫星仓表现好坏不反哺核心仓参数。
2. **仓位标尺**：卫星仓占总资金比例由用户另行决定（讨论基准 10–20%）。本课题回撤 25–30% 属正常预期，不是失败信号。
3. **简单纪律优先**：R028/R037 自适应机制两连败 vs R038 先验常数一次通过的教训直接适用——v1 参数全部取先验常数，**零调参、零搜索**，冻结后直接走验收。

### 0.2 信号选型的文献依据（为什么是动量而不是舆情）

选型结论来自 2026-07-12 的文献调研（英文学术 + 国内卖方实践），要点：

| 信号 | 证据 | 裁决 |
|------|------|------|
| **行业动量** | A 股行业层面动量显著且稳健（控制 lead-lag、一月效应、个股动量后仍在，EMFT 2011；行业动量源于对行业特定信息的反应不足）。美股行业动量（Moskowitz-Grinblatt 1999）ETF 实现约 5%/年超额。 | **v1 主信号** |
| 个股月频动量（美式 12-1） | A 股**失效甚至反转**：Liu-Stambaugh-Yuan (2019) 发现 1 个月至 5 年各回望期过去赢家均倾向跑输。 | 不用 12 月窗口；回望期取短（1M+3M） |
| 舆情/热点/关注度 | A 股存在**日频**动量（新股民注意力驱动），但**一周内反转**（Gao-Jiang-Xiong-Xiong, NBER w31839）。舆情确认之时即拥挤之刻。 | **不做买入信号**；其可交易本质由拥挤度代理（v2 否决项） |
| 换手/拥挤度 | Liu-Stambaugh-Yuan 的 A 股定制因子模型用换手率情绪因子替代动量位；银河金工行业 ETF 矩阵用"高动量 + 低拥挤"。 | v2 迭代（D2），用 `amount` 分位实现，不引入外部数据源 |
| 宏观周期时钟 | "美林电风扇"，A 股实证声名狼藉。 | 不做 |
| 北向资金流 | 监管打击假外资后预测力已衰减（BFI Chicago）。 | 不做 |

另一条影响参数的实证：A 股动量有 regime 依赖（有研究发现 2005 年前显著、其后衰减），故验收必须做起始日期敏感性 + 分段观察，且对回测总窗口较短（约 5 年）这一事实在报告中如实声明。

## 2. 策略规格 v1：`industry_momentum_rotation`

### 2.1 信号与选择规则

设 P 为后复权收盘价（`get_price_frame` 的 `adj_close`），t 为信号日（月末最后一个交易日）：

```
skip   = 5 个交易日（避开 A 股日频动量的一周内反转）
r_1m   = P[t-skip] / P[t-skip-21] - 1
r_3m   = P[t-skip] / P[t-skip-63] - 1
score  = 0.5 * r_1m + 0.5 * r_3m
```

每个月末交易日（`rebalance_frequency = monthly`，基类节奏判断，意图 T+1 即次月首个交易日执行）：

1. **合格集**：universe 中数据充足（≥ skip + 63 + 1 个对齐交易日）的资产，按 score 降序排名。
2. **绝对动量闸门**：score ≤ `abs_momentum_floor`（默认 0.0）的资产不可持有。
3. **排名缓冲带（黏性规则）**：现任成员（`state.positions` 中数量 > 0 且属于 universe 者）只要排名 ≤ `top_n + rank_buffer` 且通过闸门即留任；空出的名额由未持有资产中排名最高且通过闸门者依序补入，直至选满 `top_n` 或无合格者。
4. **权重**：每个入选资产固定 `1 / top_n`；选不满时缺额**留现金**（TargetPortfolio 权重和 < 1，引擎自然持币）。

### 2.2 交易触发（复用基类阈值带）

月末生成理论目标后走基类 `should_rebalance`：成员变化时目标差大（33pp）必然触发；成员未变时若各资产权重相对 1/top_n 的漂移都在 `rebalance_threshold` 内则**当月不交易**。这是 R038 阈值带思想在轮动语境的移植，用于压换手。

### 2.3 参数表（v1 全部先验冻结，不做搜索）

| 参数 | 默认值 | 依据 |
|------|--------|------|
| `universe` | 见 §3 定稿清单 | 准入标准硬性筛选，非收益择优 |
| `momentum_windows` | `[21, 63]` | A 股个股 12 月动量失效 → 行业取 1M+3M 短窗；等权混合 |
| `skip_days` | `5` | 日频动量一周内反转（NBER w31839） |
| `top_n` | `3` | 集中度与单行业黑天鹅的折中；行业 ETF 已消灭个股风险 |
| `rank_buffer` | `2` | 迟滞消噪，对应 Swedroe 带思想 |
| `abs_momentum_floor` | `0.0` | 动量崩溃防护的最简形式：负动量宁可持币 |
| `rebalance_threshold` | `0.05` | 与 R038 abs_band 同源的先验常数 |
| `cooldown_days` | `0` | 月频本身即冷却 |

### 2.4 已知特性（不是缺陷，验收时如实报告）

- **V 型反转踏空**：政策驱动的急涨初段（如 2024-09 式行情）策略大概率持币或持防御行业，会踏空第一波再在次月追入。这是动量策略的固有成本，不通过加机制修补（那是 R028 老路）。
- **月频滞后**：热点以周计的短题材本策略天然放弃，这是设计选择（周频的鞭打与成本实证上更差）。

## 4. 验收标准（按 AGENTS.md 全套，无豁免）

1. **pytest**：合成数据上验证——目标权重合法（非负、和 ≤ 1）；Top-N 选择正确；缓冲带黏性（排名滑到 top_n+rank_buffer 内的现任不被换出）；闸门触发时缺额留现金；月频节奏（非月末不出目标）。
2. **训练样本对比**（end 2025-06-30）：候选 `r9_rotation_industry_momentum` vs 基线 `baseline_r9_industry_equal_weight`（同池同费用月频等权）——**三滑点场景全跑**（default / stress / dynamic_participation）。
3. **起始日期敏感性**：从最早共同交易日起每 2 个自然月一个起点，截止 2025-06-30。
4. **报告指标**：年化收益、波动、最大回撤、Sharpe、换手、交易/订单/拒单数；报告需附分年度表现（观察 regime 依赖）并声明窗口较短的局限。
5. **冻结样本**（2025-07-01 之后）：仅在训练样本验收通过、参数冻结后一次性运行；见结果后不改参数。
6. **成功判据**（先验声明）：训练样本上候选相对行业等权基线年化超额 ≥ 2pp 或 Sharpe 明显占优，且回撤不显著恶化、三滑点场景下结论不翻转、敏感性排名稳定、冻结样本不打脸。不达标 → 判 research-only，撤注册，结论写 `research-dashboard/notes/`。

**基线选择说明**：对照物是"持有全部行业等权"，即"轮动选择有没有价值"这个问题本身。不与 R038 核心仓对比——两者目标不同（卫星求高收益弹性，核心求低波纪律），放一起比是范畴错误。

## 5. 迭代路线（每项独立会话/PR）

- **D1（本蓝图 v1）**：纯动量 + 缓冲带 + 闸门。先跑通、先验收。
- **D2 拥挤度否决**：`amount`（成交额）历史分位 > 0.9 的资产禁止**新开仓**（不强平现任）。需要给 `LocalCsvBarData` 加一个对称的 `get_amount_frame()`（数据 CSV 已有 amount 列，改动很小）。做成 D1 的增量对照实验：同池同参，唯一差异是否决项，按 AGENTS.md 标准判增量价值。
- **D3 货币 ETF 兜底**：闸门触发的缺额从"留现金"改为买入货币 ETF（如 511880/511990），需先入库其数据并确认分红/计价特性在引擎中的表现。回测里现金零息，此项预计增厚 ~1.5–2%/年 × 平均空仓比例，属锦上添花。
- **D4 实战接入**：训练+冻结样本双过后，建 `sim_r9_rotation_shadow` 影子组合跟跑 2–3 个月，再决定是否开 `live_r9_*` 真钱组合。完全复用主线 A 环路，无新代码。

## 6. 明确不做（本课题范围内）

- 舆情爬虫、新闻 NLP、社交媒体数据源（信号密度低、工程重、且文献方向不支持）。

- 机器学习排序、参数网格搜索（Hard Rule 4；v1 参数先验冻结是特性）。
