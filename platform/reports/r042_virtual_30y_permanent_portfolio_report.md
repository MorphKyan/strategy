# R042：虚拟 30 年国债中国永久组合研究报告

> 结论：`Research-Only`。固定 25% 权重与 5/25 阈值主候选通过训练、严格自然月起点敏感性及自身最终测试硬门槛，但两个风险平价迁移审计在最终测试均出现 Sharpe 退化，触发预先冻结的“不得宣称算法普适性”限制。没有看完最终测试后调倍数、权重、阈值或起点；六份研究配置已清理，仅保留原始产物、冻结证据和中文研究记录。

完整命令、正式/排除敏感性 manifest、冻结 YAML 与 snapshot SHA256、缓存检查、测试输出及文件清单见 `platform/reports/r042_evidence_and_command_appendix.md`。

主候选最终测试 Sharpe 差为 `-0.046/+0.006/-0.046`，均高于 `-0.10` 容忍下限；回撤最多恶化 0.33 个百分点，换手没有增加，主候选自身硬门槛通过。

最终测试债券平均绝对权重偏离：10Y 为 `1.38%~1.41%`，虚拟30Y 为 `1.07%`；最大偏离分别为 `3.67%~3.84%` 与 `2.84%~2.87%`。跳过订单为 10Y 每场景 1、虚拟30Y每场景 1~2，均无拒单或持续待成交。

但迁移审计发生明确反转：

- `risk_parity`：虚拟30Y Sharpe `1.334/1.315/1.334`，10Y 为 `1.750/1.709/1.750`；虚拟30Y回撤约 `-4.01%`，10Y约 `-1.65%`。
- `risk_parity_ewma`：虚拟30Y Sharpe `1.375/1.335/1.375`，10Y 为 `2.020/1.965/2.020`；虚拟30Y回撤约 `-4.40%`，10Y约 `-1.68%`。

两种算法最终测试均 Sharpe 退化，且回撤扩大超过 2 个百分点，触发预冻结的迁移失败条件。因此不能把训练期优势解释为“虚拟30Y替换具有算法普适性”，结论降级为 `Research-Only`。

## 六、金融解释与风险

训练期 3 倍债券袖套提高收益且没有放大组合波动，符合利率下行环境中久期贡献增强的机制；最终测试主候选仅大致持平，而两类风险平价明显退化，表明效果依赖固定资本权重和特定利率路径。固定 3 倍线性收益无法刻画久期/凸性非线性、10Y/30Y曲线扭曲、期货基差/CTD/展期/保证金/融资成本；逐日 3 倍复利还存在路径依赖与波动损耗。虚拟 OHLC、成交额、涨跌停和流动性不能代表真实 30Y ETF 或 TL 期货，实盘冲击与极端行情不可成交风险可能被低估。

建议：不合入或替换 R038 的真实 10Y 基线。若用户接受研究型影子组合，可冻结本版本做低资金规模观察，但不得据此调整 3 倍、25% 权重或 5/25 阈值；任何后续变更应作为新课题重新走完整样本隔离流程。
