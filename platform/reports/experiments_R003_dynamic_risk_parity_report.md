# 结合趋势动量与波动率靶向的动态风险预算策略实验报告 (R003)

## 1. 研究课题与假设
- **研究课题**：R003: 结合趋势动量与波动率靶向的动态风险预算策略
- **研究背景**：传统等风险平价策略（如等风险预算）在资产大下行周期中，由于没有趋势防线，容易在资产持续单边下跌时产生被动的“越跌越买”，导致组合承受不必要的深幅回撤。
- **核心假设**：
  1. 引入 **Sleeve 级别的趋势动量**：通过计算各资产过去 $N$ 日（默认 60）的趋势动量 $M_i$，动态调节各资产的风险预算 $B_i = 1.0 + \gamma \cdot M_i$（$\gamma = 1.5$ 为动量敏感度），当下行趋势确立时自动调低预算以降低暴露，避免“越跌越买”。
  2. 引入 **波动率靶向机制（Volatility Targeting）**：计算归一化权重下的组合预期年化波动率 $\sigma_{p, \text{annual}}$，当其超出设定的目标波动率 $\sigma_{\text{target}} = 8\%$ 时，按比例缩小持仓权重（$\text{leverage} = \sigma_{\text{target}} / \sigma_{p, \text{annual}}$），剩余权重留作现金防御，降低系统性风险。

## 2. 策略代码改动说明
- **修改文件**：`platform/src/platform_core/strategy.py`
- **新增策略类**：`RiskParityDynamicBudgetStrategy`
- **注册字典**：`BUILTIN_STRATEGIES` 中新增注册项 `risk_parity_dynamic_budget`
- **核心参数配置**：
  - `momentum_window`: 60 (动量计算窗口)
  - `momentum_sensitivity`: 1.5 (动量敏感度)
  - `volatility_target`: 0.08 (目标年化波动率)
  - `momentum_type`: "return" (区间收益率动量)
  - `max_leverage`: 1.0 (最大杠杆比例)
  - `budget_min`: 0.1 (最低风险预算下限，防止出现负预算)

## 5. 指标变化与物理机制分析
1. **最大回撤显著降低**：新策略的最大回撤从 **-4.37%** 降至 **-3.89%**，回撤绝对值改善 **0.48%**。这表明当组合中的高波动风险资产下行时，趋势动量动态下调了其风险预算，从而避免了传统等风险平价中“越跌越买”的恶性循环。
2. **夏普比率优化**：夏普比率从 **1.669** 提高至 **1.677**。虽然年化收益率微幅下降了 0.09%，但得益于更好的波动率与回撤控制，单位风险获得的超额回报得到了提高。
3. **波动率靶向防线有效**：Candidate 策略的平均现金占比为 **7.25%**（Baseline 仅为 2.41%），说明在整体市场波动放大的周期中，策略通过波动率靶向机制主动进行了防守减仓，自动保留现金，达到了保护大底回撤的设计初衷。
4. **降低摩擦换手率**：年化换手率从 **44.93%** 降至 **43.39%**。表明动态风险预算不仅使权重分配更平稳，而且波动靶向减少了无意义的频繁超额调仓。
