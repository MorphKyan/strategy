# R028: 基于非对称磁滞区间与系统波动触发的换手约束风险平价策略实验报告

## 1. 目标与改进构想
本课题（R028）旨在通过引入**非对称磁滞区间（Asymmetric Hysteresis Band）**与**系统波动危机触发机制（Crisis Trigger）**，对 Ledoit-Wolf 协方差收缩风险平价（或其他基准风险平价策略）的调仓流程进行精细化改造：
1. **自适应偏离阈值 (Adaptive Threshold)**：基于短期系统波动率与长期中枢波动率的比值 $R_{vol} = \sigma_{short} / \sigma_{long}$，动态缩放基础调仓阈值，实现高波动下的换手阻尼。
2. **非对称磁滞区间决策 (Asymmetric Hysteresis Band)**：
   - 对于超配资产（$curr\_weight \ge target\_weight$），为让盈利多跑一会儿并减缓调减摩擦，将容忍阈值放大至 $\theta_{up} = \theta_t \cdot (1 + \alpha)$，其中 $\alpha$ 为 `asymmetric_bias`。
   - 对于低配资产（$curr\_weight < target\_weight$），为快速捕获配置敞口防止防御失效，将容忍区间收紧为 $\theta_{down} = \theta_t \cdot (1 - \alpha)$。
3. **系统波动危机触发 (Crisis Trigger)**：当短期系统波动率暴增超过中枢的 2 倍（$vol\_ratio > 2.0$）时，强行重构风险暴露，实现危机防守。

---

## 2. 实验设置与样本切分
- **训练样本（In-Sample, IS）**：最早可用共同交易日至 `2025-06-30`。
- **最终测试样本（Out-of-Sample, OOS）**：`2025-07-01` 至 `2026-06-10`。
- **配置覆盖**：对 `platform/configs/` 下所有的平台配置进行回测。排除其中不符合 3 年最小历史约束的配置（`baseline_r7_cluster_representative_damped` 资产组合因最早共同日期到训练集末端不满三年被拒绝）。
- **起点敏感性测试**：在训练样本内，从各组合最早可用交易日起，每隔 2 个自然月（60天）生成一个 `start_date` 直至训练末期，以新策略进行跑程并统计 Sharpe 和 MDD 在各起点下的标准差（Sharpe 标准差须 < 0.25，MDD 标准差须 < 2%）。

## 4. 起点敏感性测试分析
起点敏感性测试表明：
- **测试结果全线失败**：在 20 个有效的对照配置下，新策略在各个自然月起点下展现出**极大的指标不稳定性**。
- **Sharpe Std 远超门槛**：所有配置在不同起点的 Sharpe 波动标准差在 `0.18` 至 `1.02` 之间，除单个纳指全天候外，其余全线超出 `0.25` 的波动控制红线（例如全球红利 EWMA 的 Sharpe 标准差达 1.019，美股 EWMA 的 Sharpe 标准差达 1.020）。
- **MDD Std 超限**：大部分配置的 MDD 波动标准差均突破了 `2.0%` 的防守界限（部分甚至达到 9.89% 波动率）。
- **不稳定性归因**：
  非对称磁滞机制下，在超配和低配区间所采用的容忍偏差是不等的。这虽然在特定历史回放中能获得少许的换手节约，但对历史起跑日极其敏感。不同的初始建仓日在不同的行情节点发生漂移偏离，使得后续调仓的“磁滞惯性锁死”状态存在天壤之别，导致回测路径产生了巨大的分化。由于对起跑点路径依赖度过高，该设计在面临不同的进入时点时极易失真失效，具有显著的过拟合和路径依赖风险。

## 5. IS/OOS 对比与策略衰退归因
- **训练期（IS）全线衰退**：在绝大部分资产配置包中，候选策略的 IS Sharpe 均显著差于 Baseline 策略（如在 `global_ewma` 下由 1.200 衰退至 0.844，最大回撤由 -7.20% 重挫至 -12.05%）。
- **原因分析**：
  1. **低配敞口刚性不足**：低配阈值 theta_down 变窄使得低配资产加仓极其频繁。而在市场处于趋势行情中，被低配的资产通常是高波且可能正处于强单边回调的资产，策略由于过于窄的低配容忍度频繁调仓“补仓”，在震荡与下行中遭受 whipsaw 磨损，无法有效避险。
  2. **超配阻尼导致踏空**：超配阈值 theta_up 变宽导致对于超配资产（比如美股、黄金等趋势型资产）的再平衡动作滞后。当下行行情突发且前期累积了超配时，由于宽阈值的“磁滞阻尼”限制，策略难以及时将多余的风险敞口调低，造成“利润回吐”与“回撤扩大”。

## 6. 实验结论与推荐建议
- **实验结论**：`Failed (差异不大/表现衰退)`
  该策略在训练样本内存在广泛的业绩退化，在起点敏感性测试中遭遇大面积失效（不满足波动率一致性红线），不具备全线优化的鲁棒性，且存在严重的过拟合嫌疑。
- **推荐动作**：`物理拒绝合入 platform 主干，并完全擦除代码。`
