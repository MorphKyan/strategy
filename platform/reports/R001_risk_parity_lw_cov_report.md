# 量化研究报告：R001 基于 Ledoit-Wolf 协方差收缩的风险平价策略

## 1. 研究背景与假设
传统的风险平价策略在实践中通常简化为**逆波动率（Inverse Volatility）**权重分配。这种方法虽然简单，但存在以下显著缺陷：
1. **忽视了资产间的相关性**：如果两只资产相关性极高且波动率相似，逆波动率会将等量风险预算分配给它们，导致实质上的集中暴露。
2. **样本协方差的估计噪声**：直接引入样本协方差矩阵（Sample Covariance Matrix）往往会由于历史窗口有限而带来极大的估计误差，甚至在资产较多时导致协方差矩阵不可逆或非正定。

**研究假设**：
通过引入 **Ledoit-Wolf 协方差收缩模型**，将样本协方差矩阵向常数相关系数目标矩阵进行动态收缩，不仅能够度量资产间的相关性，还能显著平滑极端噪声并保证协方差矩阵的严格正定。配合 **Cyclical Coordinate Descent (CCD)** 算法，我们可以稳健地求解真正的风险平价权重，从而优化资产组合的风险收益比，降低波动率。

---

## 2. 策略设计与代码改动
本研究在平台系统 `platform/src/platform_core/strategy.py` 中实现了 `RiskParityLWCovStrategy` 策略，并将其注册到 `BUILTIN_STRATEGIES`。

### 核心改动：
1. **Ledoit-Wolf 常数相关系数收缩估计**：
   - 算法计算样本协方差 $S$、样本相关系数均值 $\bar{\rho}$，以及渐近方差和协方差参数 $\pi$ 和 $\rho$。
   - 动态推导最优收缩比例 $\delta \in [0, 1]$，得到收缩后的协方差矩阵 $\Sigma = \delta F + (1-\delta) S$。
   - 完全基于 Numpy 和 Pandas 实现，不引入 `scikit-learn` 或 `scipy` 外部依赖。
2. **CCD 风险平价数值求解器**：
   - 采用 Cyclical Coordinate Descent 算法，以一元二次方程形式逐项循环迭代更新 $x_i$，避免了矩阵求逆，迭代 20-50 次即可达到 $10^{-6}$ 的收敛精度。
   - 自动满足权重非负 ($w_i > 0$) 且归一化后总和为 1。

## 4. 实验结论与推荐动作
1. **波动率降低与夏普提升**：候选策略在引入相关性与协方差收缩后，波动率下降了 0.10%，夏普比率从 1.6685 提升至 1.6775。表明 Ledoit-Wolf 能够更准确地反映真实的资产联动风险。
2. **更优的换手控制**：年化换手率略微降低（-1094.50），表明收缩技术不仅平滑了噪声，也增强了资产分配权重的稳定性。
3. **最大回撤保持平稳**：在回测周期内，最大回撤没有发生任何恶化，稳定在 -4.37%。
4. **推荐动作**：**接受并合并 (Accept)** 该候选策略。Ledoit-Wolf 协方差收缩显著提升了投资组合的风险控制能力，符合风险平价策略的初衷。
