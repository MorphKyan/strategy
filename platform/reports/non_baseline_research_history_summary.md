# 非基线测试与研究历史价值结果汇总报告

## 1. 概述
在风险均衡与多资产组合优化研究过程中，我们进行了多轮探索性实验和参数测试。虽然部分测试由于不符合防御性原则或存在相关性冗余，未被纳入最终的 baseline 记录中，但其回测结果和归因发现为后续研究提供了极其宝贵的“负面清单”和结构性见解。

为了维持代码库和结果目录的整洁，我们对无用的历史运行数据（如老旧的中间计算步骤、重复的参数微调运行）进行了清理，同时将其中依然具有学术和实践价值的非基线成果提炼汇总于此。

---

## 2. 多资产组合交叉回测与相关性发现（平台实验 Case 3, 4, 7）

我们在平台中对纳指、恒指、标普、沪深300/500等大类资产进行了多种非基线搭配测试，核心发现如下：

### 2.1 美股双引擎堆叠的波动放大效应（Case 3）
*   **资产组合**：沪深300 + 红利低波 + 标普500 + 纳指100 + 黄金 + 十年国债 + 豆粕 + 能化
*   **回测表现**：年化收益率 **8.73%**，年化波动率 **4.67%**，最大回撤 **-4.66%**，夏普比率 **1.8672**
*   **研究价值**：
    标普500与纳指100的日频相关性极高（**0.9025**）。在扁平风险平价组合中同时塞入这两只境外股票，相当于在海外市场进行了“双倍权重下注”，违背了分散化初衷。虽然美股近年牛市使组合收益拉升至 8.73%，但组合波动率和回撤显著放大，证明在没有进行多级分层控制前，**不应在组合中同时配置纳指与标普**。

### 2.2 恒指引入的结构性拖累与潜在分散性（Case 4）
*   **资产组合**：沪深300 + 红利低波 + 恒指（159920） + 黄金 + 十年国债 + 豆粕 + 能化
*   **回测表现**：年化收益率 **7.65%**，年化波动率 **4.37%**，最大回撤 **-3.96%**，夏普比率 **1.7498**
*   **研究价值**：
    港股市场在 2022-2024 年受地缘政治和流动性挤压，跌幅居全球主要指数前列，导致该组合表现偏弱。然而，从相关性分析来看，恒指与美股（标普/纳指）的相关性仅为 **0.34-0.36**，而与国内沪深300的相关性高达 **0.7147**。这说明恒指难以作为境外权益的独立风险分散工具，而是更偏向于作为 A 股的低估值、高弹性替代通道。

### 2.3 全球全资产扁平拥挤的失败案例（Case 7）
*   **资产组合**：沪深300 + 中证500 + 红利低波 + 标普 + 纳指 + 恒指 + 黄金 + 十年国债 + 三只商品期货
*   **回测表现**：年化收益率 **9.06%**，年化波动率 **5.72%**，最大回撤 **-5.06%**，夏普比率 **1.5838**
*   **研究价值**：
    这是证明**“扁平风险平价资产拥挤”**的经典反面教材。由于股票和商品细分项过多（共9只风险资产，仅1只低波国债），算法在扁平结构下对高波股票类别分配了过多的累积风险预算，稀释了十年国债的防守比重。虽然获得了 9.06% 的名义年化收益，但波动率大幅失控（5.72%），最大回撤突破 -5.06%，夏普比率退化至 1.5838。本实验强力支持了**未来必须采用多级分层风险平价（HRP）**的策略架构。

## 3. 无债风险平价组合实验（`no_bond_risk_parity_baskets`）

### 3.1 实验背景
为了测试在没有“基石资产”（十年国债）提供保护的极端通胀或全球衰退场景下，风险平价组合仅靠权益、黄金和商品能否通过自身的低相关性提供绝对收益。

### 3.2 回测表现
*   **测试组合**：黄金 + 沪深300 + 豆粕 + 能化（无债券防线）
*   **研究价值**：
    在失去国债平滑波动的保护后，组合的权重被迫由黄金（波幅~15%）和商品（波幅~18-22%）主导。在 2022 年大宗商品暴涨期间，组合获得了极佳的抗通胀溢价，但随着 2023 年商品高位回落，组合波动率翻倍，回撤大幅上升。
    **核心结论**：债券袖子不仅提供名义利息，更重要的是在风险平价中作为杠杆分配和整体波动的压舱石。无债组合仅在供给端受限的极端恶性通胀期具有阶段性战术配置价值，不适合作为长期全天候战略底仓。

## 4. 同起点回测诊断实验（`agri_energy_same_start_20260531`）

### 4.1 实验背景
标准研究实验中，由于商品期货 ETF 历史较短（2022年4月成立），导致候选组合的回测时长被迫缩短，而 baseline 回测期长达 8 年。这种“非对齐起点”的比较容易高估商品袖子的阶段性贡献。

### 4.2 诊断结论
*   **对齐区间**：`2022-05-31` 至 `2026-05-19`
*   **诊断结果**：
    在严格对齐起点和交易成本的前提下，引入商品袖子的组合表现依然显著优于只含股债金的国内基准。这证明**商品袖子的超额收益并不是特定短样本的幸存者偏差**，而是由于其在 2022-2024 年与国内股债呈现出的负相关/弱相关关系，真实地改善了组合的有效前沿。

## 6. 自适应回撤恢复算法消融实验与算法 A 最优落地（2026-06-01）

### 6.1 实验背景
针对前代经典 EWMA 协方差估算模型顺周期时滞导致的“大跌初期慢半拍，反弹初期空守候”问题，我们设计了自适应波动率缩放算法。然而，由于该算法对波动过度敏感，导致红利、低波等本就极低波动的均值回归组合发生了显著的假反弹摩擦损耗（Whipsaw Loss）。为此，我们设计了非线性激活阈值（算法 A）、趋势均线确认过滤器（算法 B）及其双重组合（组合 C）的消融对照方案。

### 6.2 诊断与消融实验结论
在公共区间（2022-05-31 至 2026-06-01）对 27 个资产配置策略进行了 108 次交叉回测后，统计显示：
*   **算法 A（仅非线性激活阈值 $\tau=2.5\%$）** 全面胜出：平均夏普比率由基线的 2.1819 提升至 **2.1998**，平均最大回撤由基线的 -3.62% 显著收缩至 **-3.43%**，且回撤优化胜出率达到了惊人的 **74.07%**，夏普胜出率达 **59.26%**。
*   **算法 B（仅趋势均线过滤器）** 遭遇严重时滞：由于 10 日均线的滞后性，在反弹最强烈的首段踏空了涨幅，夏普（2.1813）甚至逊于基线。
*   **组合 C（阈值+过滤器）** 受到时滞负面影响，表现（夏普 2.1920，回撤 -3.54%）虽优于基准但弱于算法 A。这说明复杂的双重过滤器带来了多层滞后噪声，验证了“如无必要，勿增实体”的奥卡姆剃刀原理。

### 6.3 最优算法 A 固化与通用化落地
为了使算法 A 的成果独立于单一组合配置，我们已将其最优参数固化为平台底层策略 `RiskParityEWMADrawdownRecoveryStrategy` (`"risk_parity_ewma_dd_recovery"`) 的全局默认行为：
*   **默认大跌惩罚 $\alpha$** = `1.0`
*   **默认反弹加速 $\beta$** = `2.0`
*   **默认回撤观测窗口 $W$** = `30` 交易日
*   **默认非线性激活阈值 $\tau$** = `0.025` ($2.5\%$ )
*   **默认不启用趋势过滤器** (`rebound_filter_window` = `0`)

通过此次底层策略的默认参数改造，所有现有或未来的策略组合配置，只要在 yaml 中选用了 `"risk_parity_ewma_dd_recovery"` 策略且不配置特定参数，均会自动应用该套基于消融实验验证过的最优非线性激活阈值配置，达成了系统级的算法配置共享。
