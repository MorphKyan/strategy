### R035: 基于 Gerber 稳健相关性与 Ledoit-Wolf 混合收缩协方差估计的风险平价策略
- **研究背景**：
  - Gerber 统计量 (R020) 在消除白噪声、防 whipsaw 调仓磨损方面效果极佳。然而，在高维多资产（如跨境多资产组合）或极端行情下，纯 Gerber 矩阵可能面临条件数过大、求逆困难甚至非正定的问题，而 Ledoit-Wolf 收缩 (R001) 能通过向目标矩阵收缩显著改善矩阵的条件数。本项目拟将 Gerber 稳健相关性与 Ledoit-Wolf 的线性收缩方法融合，以增强协方差在极端行情下的估计稳健性与正定性。
- **算法原理**：
  1. 通过 Gerber 阈值法（如设置比例因子 c=0.5）计算资产收益率的稳健相关系数矩阵 $R_{Gerber}$。
  2. 使用稳健的标准差估计构建对角阵 $D$，得到初始稳健协方差估计 $\Sigma_{Gerber} = D R_{Gerber} D$。
  3. 实施 Ledoit-Wolf 线性收缩，将 $\Sigma_{Gerber}$ 向常数相关矩阵（Constant Correlation Target）或恒等矩阵收缩：$\Sigma^* = \delta F + (1-\delta) \Sigma_{Gerber}$，其中 $F$ 为收缩目标矩阵，$\delta$ 为通过解析公式估计的最优收缩强度。
  4. 将收缩后的 $\Sigma^*$ 作为协方差矩阵输入风险平价优化求解器，得出最终资产权重。
- **预期改善**：
  - 融合 Gerber 的降噪避险能力与 Ledoit-Wolf 提升特征值稳定性的特性。在跨境及大类资产组合下，相比单一 Gerber 或 LW 协方差估计，可降低极端波动下的调仓剧烈度，提高 Sharpe 比率并平滑换手。
- **推荐回测配置**：
  - `baseline_r3_global_nasdaq_all_weather_ewma` (10y国债替代为 511260_3X)
  - `baseline_r1_domestic_rolling` (10y国债替代为 511260_3X)

  - 训练/研究样本区间：截至 `2025-06-30`（含）的历史数据。
  - 最终测试样本区间：自 `2025-07-01`（含）起的数据。不得在研究阶段使用最终测试样本数据进行优化、参数微调或表现判断。
  - 共同训练样本长度大于三年：确认所选配置最早可用共同交易日到训练样本末日（最晚 `2025-06-30`） the 训练样本长度大于三年（本配置已确认 511260_3X 的最早有行情日期）。

### R036: 基于 GARCH-EWMA 混合滤波与成分风险贡献限额的自适应风险平价策略
- **研究背景**：
  - 传统的风险平价在计算条件协方差时，如果直接使用时变 GARCH 估计 (R017, R007)，会因为模型对短期收益波动的极端敏感性导致协方差估计频繁巨幅波动，从而使优化权重频繁大幅跳动，引起换手率暴涨。为了克服这一问题，本策略拟引入 GARCH 条件方差的 EWMA 滤波平滑，并在风险平价求解器中引入硬性或软性的成分风险贡献边界（Risk Contribution Bounds），既保留 GARCH 捕捉长线波动聚集与肥尾风险的优势，又大幅限制换手磨损。
- **算法原理**：
  1. 通过单变量 GARCH(1,1) 模型拟合各资产的日收益率，提取条件方差序列 $\sigma^2_{i, GARCH, t}$。
  2. 对 GARCH 条件方差应用指数加权移动平均 (EWMA) 进行二次滤波平滑：$\hat{\sigma}^2_{i, t} = \alpha \sigma^2_{i, GARCH, t} + (1-\alpha) \hat{\sigma}^2_{i, t-1}$，其中 $\alpha \in [0.05, 0.2]$，用以过滤短期估计噪声。
  3. 构建时变相关系数矩阵并结合平滑方差预测构建协方差矩阵。
  4. 在风险平价优化阶段，引入成分风险贡献限制条件：$RC_i \in [RC_{target, i} - \epsilon, RC_{target, i} + \epsilon]$，防止因条件方差突变导致资产风险预算分配被瞬时压低为零，并对权重变化的二次偏差进行惩罚以限制高频换手。
- **预期改善**：
  - 有效过滤时变波动率估计的日度随机噪声，平抑权重抖动，预期能解决 R017/R007 等时变协方差策略换手率暴涨的痛点，同时能比传统滚动协方差更灵敏地识别市场极端下行风险。
- **推荐回测配置**：
  - `baseline_r1_domestic_rolling` (10y国债替代为 511260_3X)
  - `baseline_r3_global_nasdaq_all_weather_ewma` (10y国债替代为 511260_3X)
- **研究样本与测试约束**：
  - 训练/研究样本区间：截至 `2025-06-30`（含）的历史数据。
  - 最终测试样本区间：自 `2025-07-01`（含）起的数据。不得在研究阶段使用最终测试样本数据进行优化、参数微调或表现判断。
  - 共同训练样本长度大于三年：确认所选配置最早可用共同交易日到训练样本末日（最晚 `2025-06-30`）的训练样本长度大于三年。
- **起点敏感性测试要求**：
  - 在不触碰最终测试样本的前提下，从配置最早可用共同交易日开始，至训练样本末日 `2025-06-30`，每隔 2 个自然月生成一个 `start_date`，独立运行回测并评估 GARCH-EWMA 参数稳定性。
- **合入验收门槛（合格标准）**：
  - 在扣除万分之五双边交易成本后，训练集内全线配置的 Sharpe 比率相比 Ledoit-Wolf 基线有改善或不退化，且年化双边换手率必须相比传统 GARCH-LW 策略降低 40% 以上，相比 LW 基线换手率变化控制在 $\pm 10\%$ 以内。
  - 起点敏感性测试中显示 Sharpe 波动标准差 < 0.22，最大回撤波动 < 2.0%。
  - 最终测试阶段（OOS，`2025-07-01` 之后）的 Sharpe 不低于基线，最大回撤无明显恶化，换手控制维持 IS 阶段的优化水平。

### R037: 基于波动率半衰期衰减动态阈值触发的自适应再平衡风险平价策略
- **研究背景**：
  - R016 (自适应偏离阈值+系统波动触发) 采用静态的偏离阈值取得了极好的降低换手效果，而 R026/R028 等利用刚性惩罚和偏差惩罚项的策略都因为权重锁死而失败。本策略拟对 R016 进行拓展，利用资产波动率的“半衰期”特征动态调整调仓偏离阈值。当市场波动率较高且衰减缓慢（波动半衰期长/高波震荡）时，系统容忍度增加（调仓阀值变宽）以防 whipsaw；当市场处于波动尖峰但衰减极快（半衰期短/避险切换）时，调仓阀值变窄，加速避险调仓，实现比 R016 更加自适应的精细化再平衡控制。
- **算法原理**：
  1. 通过 AR(1) 过程或相关性建模对组合及资产的波动率自相关系数进行估计，计算各资产条件波动的半衰期（Half-Life） $\tau_i = -\ln(2) / \ln(\rho_i)$，其中 $\rho_i$ 为波动率的自相关系数。
  2. 根据组合持仓加权半衰期 $\bar{\tau}_t$ 动态调整偏离度再平衡触发门槛 $\theta_t = \theta_{base} \times \exp(-\gamma / \bar{\tau}_t)$，其中 $\theta_{base}$ 为基础偏离触发门槛，$\gamma$ 为调节系数。
  3. 计算当前实际权重与理论风险平价权重的偏离度（如 $L_1$ 或 $L_2$ 距离）。当且仅当偏离度超过动态阈值 $\theta_t$ 时，才触发权重再平衡，且只调平偏离幅度的 80%（平滑再平衡）。
- **预期改善**：
  - 避免在市场无方向高频震荡时频繁调仓产生 whipsaw 摩擦损耗，同时确保在重大突发危机导致波动瞬间跃升、半衰期剧烈缩短时，能够极其敏捷地进行避险资产（如模拟30年国债期货 `511260_3X`）切换，提升 Sharpe 并降低换手。
- **推荐回测配置**：
  - `baseline_r1_domestic_rolling` (10y国债替代为 511260_3X)
  - `baseline_r2_global_dividend_ewma` (10y国债替代为 511260_3X)

### R040: 基于层级等风险贡献 (HERC) 与簇间风险预算约束的多资产风险平价策略
- **研究背景**：
  - 平台历史研究显示，R008 的 HRP 和 R019 的聚类代表筛选能够利用相关结构降低协方差不稳定和冗余资产拥挤；但 R019 仍偏向“选代表资产”，不是直接约束每个资产大类或相关簇的总风险预算。当前多资产配置中，权益、商品、黄金、30y 模拟国债等 sleeve 的资产数量不均衡，普通逐资产风险平价可能让资产数量更多的同类风险因子获得更高组合风险占比，削弱债券和黄金的防御权重。
  - Raffinot 的 Hierarchical Equal Risk Contribution (HERC) 研究提出在层级簇内外分配资本与风险，目标是同时利用相关结构与等风险贡献思想；Lopez de Prado 的 HRP 与 Nested Clustered Optimization (NCO) 研究指出，聚类能降低均值-方差类优化的估计不稳定、集中度和样本外退化风险。参考链接：[HERC SSRN](https://papers.ssrn.com/sol3/Delivery.cfm/SSRN_ID3237540_code2270025.pdf?abstractid=3237540&mirid=1)、[HRP SSRN](https://papers.ssrn.com/sol3/papers.cfm?abstract_id=2708678)、[NCO SSRN](https://papers.ssrn.com/sol3/papers.cfm?abstract_id=3469961)。
- **算法原理**：
  1. 在训练样本内（最晚截至 `2025-06-30`）使用资产 sleeve 标签或基于相关距离的层级聚类，将资产划分为权益、债券、黄金、商品、跨境权益等簇；簇数量和聚类阈值不得使用 `2025-07-01` 及之后数据选择。
  2. 先在簇间构建簇收益序列和簇协方差矩阵，执行等风险贡献分配，得到每个簇的总风险预算上限和目标权重。
  3. 在每个簇内使用现有 `risk_parity_lw_cov`、`risk_parity_ewma_cov` 或 Gerber 风险平价进行子组合权重分配，再将簇间权重与簇内权重相乘得到最终资产权重。
  4. 可加入轻量的簇权重平滑或再平衡阈值，但不得引入大规模参数搜索；任何阈值范围必须在研究启动前冻结，并仅用训练样本验证。
- **预期改善**：
  - 在不扩大资产池和不修改基准 benchmark 的前提下，减少“同类资产越多、总风险预算越大”的结构性偏差，特别是缓解全球权益或商品 sleeve 对组合风险预算的挤占。
  - 相比逐资产风险平价，预期能在 `baseline_r2_global_ewma`、`baseline_r3_global_nasdaq_all_weather_ewma` 和 `baseline_r7_cluster_representative_damped` 等多资产配置中降低最大回撤、降低起点敏感性，并保持或降低年化双边换手率。
- **推荐回测配置**：
  - `baseline_r2_global_ewma` (10y国债替代为 `511260_3X`)
  - `baseline_r3_global_nasdaq_all_weather_ewma` (10y国债替代为 `511260_3X`)
  - `baseline_r7_cluster_representative_damped` (用于和已有聚类代表策略对照)
- **研究样本与测试约束**：
  - 训练/研究样本区间：截至 `2025-06-30`（含）的历史数据。
  - 最终测试样本区间：自 `2025-07-01`（含）起的数据。不得在形成假设、簇定义、参数范围、ETF 排名或研究结论时使用最终测试样本数据。
  - 共同训练样本长度必须大于三年；若任一推荐配置的共同可用历史不足三年，必须剔除该配置或停止输出候选配置。
  - 回测前必须按 AGENTS.md 验证所需 symbol 数据对齐且最新本地日期距当前日期不超过 7 个自然日；若数据过期，必须先同步并复核，失败则停止。
- **起点敏感性测试要求**：
  - 在不触碰最终测试样本的前提下，从每个配置的最早共同可用交易日开始，到 `2025-06-30` 为止，每隔 2 个自然月生成一个 `start_date` 并独立回测。
  - 必须报告 Sharpe、年化收益、最大回撤、年化双边换手率、交易笔数、订单数和拒单数在不同起点下的稳定性，并检查簇划分是否因起点变化而过度跳变。
- **合入验收门槛（合格标准）**：
  - 训练样本内相比 `risk_parity_lw_cov`、`risk_parity_ewma_cov` 或已合入的 HRP/聚类代表基线，主要多资产配置 Sharpe 不得退化；若 Sharpe 持平，则最大回撤至少收窄 10% 或起点敏感性显著下降。
  - 起点敏感性测试中 Sharpe 标准差应低于 0.25，最大回撤波动不超过 2%，且簇间总风险预算不得出现单一簇长期超过 55% 的拥挤现象。
  - 年化双边换手率相比相关基线增幅不得超过 15%；若换手率上升，必须有更高 Sharpe 或更低最大回撤作为明确补偿。
  - 最终测试阶段只能在候选算法、簇规则、参数范围和验收门槛冻结后运行；`2025-07-01` 之后的结果不得反向修改候选设定。
