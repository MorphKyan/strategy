### R035: 基于 Gerber 稳健相关性与 Ledoit-Wolf 混合收缩协方差估计的风险平价策略
- **研究背景**：
  - Gerber 统计量 (R020) 在消除白噪声、防 whipsaw 调仓磨损方面效果极佳。然而，在高维多资产（如跨境多资产组合）或极端行情下，纯 Gerber 矩阵可能面临条件数过大、求逆困难甚至非正定的问题，而 Ledoit-Wolf 收缩 (R001) 能通过向目标矩阵收缩显著改善矩阵的条件数。本项目拟将 Gerber 稳健相关性与 Ledoit-Wolf 的线性收缩方法融合，以增强协方差在极端行情下的估计稳健性与正定性。
- **算法原理**：
  1. 通过 Gerber 阈值法（如设置比例因子 c=0.5）计算资产收益率的稳健相关系数矩阵 $R_{Gerber}$。
  2. 使用稳健的标准差估计构建对角阵 $D$，得到初始稳健协方差估计 $\Sigma_{Gerber} = D R_{Gerber} D$。
  3. 实施 Ledoit-Wolf 线性收缩，将 $\Sigma_{Gerber}$ 向常数相关矩阵（Constant Correlation Target）或恒等矩阵收缩：$\Sigma^* = \delta F + (1-\delta) \Sigma_{Gerber}$，其中 $F$ 为收缩目标矩阵，$\delta$ 为通过解析公式估计的最优收缩强度。
  4. 将收缩后的 $\Sigma^*$ 作为协方差矩阵输入风险平价优化求解器，得出最终资产权重。
- **预期改善**：
  - 融合 Gerber 的降噪避险能力与 Ledoit-Wolf 提升特征值稳定性的特性。在跨境及大类资产组合下，相比单一 Gerber 或 LW 协方差估计，可降低极端波动下的调仓剧烈度，提高 Sharpe 比率并平滑换手。
- **推荐回测配置**：
  - `baseline_r3_global_nasdaq_all_weather_ewma` (10y国债替代为 511260_3X)
  - `baseline_r1_domestic_rolling` (10y国债替代为 511260_3X)
- **研究样本与测试约束**：
  - 训练/研究样本区间：截至 `2025-06-30`（含）的历史数据。
  - 最终测试样本区间：自 `2025-07-01`（含）起的数据。不得在研究阶段使用最终测试样本数据进行优化、参数微调或表现判断。
  - 共同训练样本长度大于三年：确认所选配置最早可用共同交易日到训练样本末日（最晚 `2025-06-30`） the 训练样本长度大于三年（本配置已确认 511260_3X 的最早有行情日期）。

### R036: 基于 GARCH-EWMA 混合滤波与成分风险贡献限额的自适应风险平价策略
- **研究背景**：
  - 传统的风险平价在计算条件协方差时，如果直接使用时变 GARCH 估计 (R017, R007)，会因为模型对短期收益波动的极端敏感性导致协方差估计频繁巨幅波动，从而使优化权重频繁大幅跳动，引起换手率暴涨。为了克服这一问题，本策略拟引入 GARCH 条件方差的 EWMA 滤波平滑，并在风险平价求解器中引入硬性或软性的成分风险贡献边界（Risk Contribution Bounds），既保留 GARCH 捕捉长线波动聚集与肥尾风险的优势，又大幅限制换手磨损。
- **算法原理**：
  1. 通过单变量 GARCH(1,1) 模型拟合各资产的日收益率，提取条件方差序列 $\sigma^2_{i, GARCH, t}$。
  2. 对 GARCH 条件方差应用指数加权移动平均 (EWMA) 进行二次滤波平滑：$\hat{\sigma}^2_{i, t} = \alpha \sigma^2_{i, GARCH, t} + (1-\alpha) \hat{\sigma}^2_{i, t-1}$，其中 $\alpha \in [0.05, 0.2]$，用以过滤短期估计噪声。
  3. 构建时变相关系数矩阵并结合平滑方差预测构建协方差矩阵。
  4. 在风险平价优化阶段，引入成分风险贡献限制条件：$RC_i \in [RC_{target, i} - \epsilon, RC_{target, i} + \epsilon]$，防止因条件方差突变导致资产风险预算分配被瞬时压低为零，并对权重变化的二次偏差进行惩罚以限制高频换手。
- **预期改善**：
  - 有效过滤时变波动率估计的日度随机噪声，平抑权重抖动，预期能解决 R017/R007 等时变协方差策略换手率暴涨的痛点，同时能比传统滚动协方差更灵敏地识别市场极端下行风险。
- **推荐回测配置**：
  - `baseline_r1_domestic_rolling` (10y国债替代为 511260_3X)
  - `baseline_r3_global_nasdaq_all_weather_ewma` (10y国债替代为 511260_3X)
- **研究样本与测试约束**：
  - 训练/研究样本区间：截至 `2025-06-30`（含）的历史数据。
  - 最终测试样本区间：自 `2025-07-01`（含）起的数据。不得在研究阶段使用最终测试样本数据进行优化、参数微调或表现判断。
  - 共同训练样本长度大于三年：确认所选配置最早可用共同交易日到训练样本末日（最晚 `2025-06-30`）的训练样本长度大于三年。
- **起点敏感性测试要求**：
  - 在不触碰最终测试样本的前提下，从配置最早可用共同交易日开始，至训练样本末日 `2025-06-30`，每隔 2 个自然月生成一个 `start_date`，独立运行回测并评估 GARCH-EWMA 参数稳定性。
- **合入验收门槛（合格标准）**：
  - 在扣除万分之五双边交易成本后，训练集内全线配置的 Sharpe 比率相比 Ledoit-Wolf 基线有改善或不退化，且年化双边换手率必须相比传统 GARCH-LW 策略降低 40% 以上，相比 LW 基线换手率变化控制在 $\pm 10\%$ 以内。
  - 起点敏感性测试中显示 Sharpe 波动标准差 < 0.22，最大回撤波动 < 2.0%。
  - 最终测试阶段（OOS，`2025-07-01` 之后）的 Sharpe 不低于基线，最大回撤无明显恶化，换手控制维持 IS 阶段的优化水平。

### R037: 基于波动率半衰期衰减动态阈值触发的自适应再平衡风险平价策略
- **研究背景**：
  - R016 (自适应偏离阈值+系统波动触发) 采用静态的偏离阈值取得了极好的降低换手效果，而 R026/R028 等利用刚性惩罚和偏差惩罚项的策略都因为权重锁死而失败。本策略拟对 R016 进行拓展，利用资产波动率的“半衰期”特征动态调整调仓偏离阈值。当市场波动率较高且衰减缓慢（波动半衰期长/高波震荡）时，系统容忍度增加（调仓阀值变宽）以防 whipsaw；当市场处于波动尖峰但衰减极快（半衰期短/避险切换）时，调仓阀值变窄，加速避险调仓，实现比 R016 更加自适应的精细化再平衡控制。
- **算法原理**：
  1. 通过 AR(1) 过程或相关性建模对组合及资产的波动率自相关系数进行估计，计算各资产条件波动的半衰期（Half-Life） $\tau_i = -\ln(2) / \ln(\rho_i)$，其中 $\rho_i$ 为波动率的自相关系数。
  2. 根据组合持仓加权半衰期 $\bar{\tau}_t$ 动态调整偏离度再平衡触发门槛 $\theta_t = \theta_{base} \times \exp(-\gamma / \bar{\tau}_t)$，其中 $\theta_{base}$ 为基础偏离触发门槛，$\gamma$ 为调节系数。
  3. 计算当前实际权重与理论风险平价权重的偏离度（如 $L_1$ 或 $L_2$ 距离）。当且仅当偏离度超过动态阈值 $\theta_t$ 时，才触发权重再平衡，且只调平偏离幅度的 80%（平滑再平衡）。
- **预期改善**：
  - 避免在市场无方向高频震荡时频繁调仓产生 whipsaw 摩擦损耗，同时确保在重大突发危机导致波动瞬间跃升、半衰期剧烈缩短时，能够极其敏捷地进行避险资产（如模拟30年国债期货 `511260_3X`）切换，提升 Sharpe 并降低换手。
- **推荐回测配置**：
  - `baseline_r1_domestic_rolling` (10y国债替代为 511260_3X)
  - `baseline_r2_global_dividend_ewma` (10y国债替代为 511260_3X)
- **研究样本与测试约束**：
  - 训练/研究样本区间：截至 `2025-06-30`（含）的历史数据。
  - 最终测试样本区间：自 `2025-07-01`（含）起的数据。不得在研究阶段使用最终测试样本数据进行优化、参数微调或表现判断。
  - 共同训练样本长度大于三年：确认所选配置最早可用共同交易日到训练样本末日（最晚 `2025-06-30`）的训练样本长度大于三年。
- **起点敏感性测试要求**：
  - In-Sample 期间从配置最早可用共同交易日开始，至训练样本末日 `2025-06-30`，每隔 2 个自然月生成一个 `start_date`，进行回测并检查在不同起点下动态阀值的响应平滑性。
- **合入验收门槛（合格标准）**：
  - 在扣除万分之五双边交易成本后，训练集内全线配置的 Sharpe 比率相比 R016 自适应偏离基准不退化或有所提升，且年化双边换手率平均降低 15% 以上，调仓交易笔数降低 25% 以上。
  - 起点敏感性测试中，各个起跑点的 Sharpe 波动标准差 < 0.20，最大回撤波动 < 1.5%。
  - 最终测试阶段（OOS，`2025-07-01` 之后）的 Sharpe 不低于基线，最大回撤相比基线无显著恶化，且换手率维持在 R016 的 85% 以下。

### R040: 基于层级等风险贡献 (HERC) 与簇间风险预算约束的多资产风险平价策略

  - `baseline_r3_global_nasdaq_all_weather_ewma` (10y国债替代为 `511260_3X`)

### R041: 基于最小扭转有效押注数 (Minimum-Torsion ENB) 约束的风险预算拥挤度控制
- **研究背景**：
  - 平台已完成的 R008、R019 与正在推进的 R040 都从层级聚类、代表资产或簇风险预算角度处理同源资产拥挤问题，但这些方法主要依赖资产标签或相关距离，并不直接回答“组合实际押注在多少个彼此不相关的风险来源上”。在 `baseline_r2_global_ewma`、`baseline_r3_global_nasdaq_all_weather_ewma`、`baseline_us_blend_ewma` 等配置中，标普、纳指、A 股、商品和黄金的名义资产数量不等，逐资产风险平价仍可能形成隐含风险押注集中。
  - Meucci 的 `Managing Diversification` 提出用熵定义的 `Effective Number of Bets` 衡量分散度；Deguest、Martellini、Meucci 的 `Risk Budgeting and Diversification Based on Optimized Uncorrelated Factors` 进一步提出 `Minimum-Torsion Bets`，用“最接近原始资产、但彼此不相关”的押注来度量和管理组合分散度。参考链接：[Meucci Managing Diversification](https://papers.ssrn.com/sol3/papers.cfm?abstract_id=1358533)、[Minimum-Torsion Bets SSRN](https://papers.ssrn.com/sol3/papers.cfm?abstract_id=2276632)。
- **算法原理**：
  1. 在每个再平衡日，仅使用截至当日且不晚于训练样本末日 `2025-06-30` 的历史收益率估计协方差矩阵，并由相关矩阵构造 minimum-torsion 变换，得到尽量贴近原始资产的非相关押注。
  2. 将现有基线策略（优先 `risk_parity_lw_cov`、`risk_parity_ewma_cov`、`risk_parity_gerber` 或 `cluster_representative_damped_risk_parity`）生成的目标权重投影到 minimum-torsion 押注空间，计算押注贡献分布和归一化 `ENB/N`。
  3. 若 `ENB/N` 低于冻结阈值（建议候选范围：`0.45`、`0.55`、`0.65`，研究启动后不得扩展），则对目标权重施加轻量拥挤度惩罚或线性缩放，使单一 minimum-torsion 押注贡献不超过冻结上限（建议 `35%` 或 `45%`）；不得进行大规模参数搜索。
  4. 为降低交易摩擦，候选只能在原始目标权重与 ENB 修正权重之间做一次性混合（建议混合系数 `0.25` 或 `0.50`），并沿用平台既有交易成本、订单、成交和拒单统计。
- **预期改善**：
  - 相比单纯逐资产风险平价，该方向预期能识别并抑制“名义多资产、实际少数风险押注”的配置拥挤，尤其是美股双指数、A 股/港股联动、商品同源风险在协方差矩阵中的隐含集中。

  - `baseline_r2_global_ewma`

  - `baseline_r7_cluster_representative_damped`
- **研究样本与测试约束**：
  - 训练/研究样本区间：截至 `2025-06-30`（含）。
  - 最终测试样本区间：自 `2025-07-01`（含）起。不得在形成假设、阈值范围、混合系数、推荐配置或研究结论时使用最终测试样本数据。
  - 回测前必须按 `AGENTS.md` 验证所需 symbol 数据对齐且最新本地日期距当前日期不超过 7 个自然日；若数据过期，必须先同步并复核，失败则停止。
  - 任一输出候选配置的最早共同可用交易日至 `2025-06-30` 的训练样本长度必须大于三年；不足三年的配置必须剔除。
- **起点敏感性测试要求**：

### R043: Gerber 风险平价的长久期趋势保护策略
- **研究背景与可证伪假设**：
  - 本课题不使用 R042 的各25%永久组合，也不以 `2025-07-01` 及以后表现形成假设。基础组合固定为现有 `baseline_risk_parity_gerber.yaml`：`510300.SH`、`518880.SH`、`511260_3X.SH`，资本权重继续由已合入的 `risk_parity_gerber` 计算。
  - Moskowitz、Ooi、Pedersen 的时间序列动量研究在股票指数、外汇、商品与债券期货中记录了过去1至12个月收益的持续性；本课题据此预先固定12个月近似尺度，而不在仓库数据上搜索窗口。来源：[Time Series Momentum](https://w4.stern.nyu.edu/facdir/lpederse/papers/TimeSeriesMomentum.pdf)。Hurst、Ooi、Pedersen 的长期样本研究为跨市场趋势跟随的分散价值提供补充依据：[A Century of Evidence on Trend-Following Investing](https://research.cbs.dk/en/publications/a-century-of-evidence-on-trend-following-investing/)。
  - 可证伪假设是：虚拟30年国债的高久期会放大利率趋势，因此仅在债券自身长期趋势不利时机械降低久期资本暴露，并将释放资金留为现金，可能比无条件 Gerber 风险平价降低回撤；若其造成趋势转折滞后、现金拖累或阈值附近反复切换，则应由训练、起点敏感性和执行指标判定失败。
- **唯一候选与回测前冻结规则**：
  1. 基础策略、资产与参数完全继承 `baseline_risk_parity_gerber.yaml`：`rolling_window=120`、`min_periods=20`、`gerber_c=0.5`、`rebalance_frequency=monthly`、`rebalance_threshold=0.05`、`rebalance_relative_threshold=0.25`。不得根据回测修改 Gerber 参数、资产池或交易阈值。
  2. 趋势信号只使用未杠杆的 `511260.SH` 后复权收盘价：在月度再平衡日，以前一交易日为止的252个交易日总收益为信号；严格禁止使用当日收盘后信息生成当日成交目标。若可用历史不足252日，沿用基础 Gerber 权重但将虚拟债 multiplier 设为 `1/3`，不得另设 warm-up 参数。
  3. 若252日总收益严格大于0，虚拟债 multiplier 固定为 `1`；否则固定为 `1/3`。最终债券目标为 `w_bond_final = multiplier * w_bond_gerber`，股票和黄金目标不放大，释放的目标权重保留为现金。`1/3` 只来自用户预先验证并冻结的“虚拟30年债日收益约等于10年债3倍”关系，不得搜索其他档位。
  4. 信号每月只判定一次，不增加短周期确认、移动平均组合、波动率目标、宏观状态、止损、滞后带或平滑系数；禁止在训练后追加规则。候选必须通过 `Strategy.generate_targets(context)` 增量实现。
  5. 主基线固定为未经趋势保护的 `baseline_risk_parity_gerber.yaml`。机制审计只允许增加一个“信号与实际档位/现金占比”日志，不得另建参数候选或根据表现挑选子区间。
- **数据与代理风险检查**：
  - 回测前核验 `510300`、`518880`、`511260`、`511260_3X` 的最新日期、重复日期、后复权链与交易日对齐；最新日期距研究运行日不得超过7个自然日。若过期，先同步并从最新 `511260` 重建 `511260_3X`，同步或对齐失败即停止。
